# YT Scheduler

## 概要

システム手帳のこのタイプのリフィル
(1ページで一週間)
の使い勝手をイメージして作りました。

![](docs/refill1.jpg)
  
* 項目や種別などを詳細に分けすぎず、柔軟な使い勝手
  - タイトルに「!」をつけると「重要」の意味
  - タイトルに「×」をつけると、「取り消し」の意味
  - タイプを「□」にすると、「ToDo」項目と見なす

* スマホの使い勝手を活かす。
  - 無限にスクロール
  - 検索やフィルター機能の充実
